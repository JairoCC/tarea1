# Listar instancias EC2 con boto3

## Descripción
Script en Python que lista las instancias de EC2 usando boto3.

## Requisitos
- Python 3
- boto3
- Cuenta de AWS
- Credenciales configuradas (`aws configure`)

## Instalación
pip install boto3

## Uso
python list_ec2.py

## Nota
La región configurada debe coincidir con la de las instancias.
